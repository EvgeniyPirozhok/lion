General Information

Software requirements

  • Sublime Text 3 used to edit the .HTML, .JS, .CSS and .SCSS files.
  • Grunt used as precompiler for SCSS to CSS convert.

The following scheme displays the Snippet files structure.

  • Documentation - contains documentation on form customization and support
  • Notification - contains the main files of the notification design. which will be uploaded on server.
    • CSS - contains all Stylesheets used for UI.
    • Fonts - contains Bootstrap icon files.
    • Images - contains all the images.
    • Js - contains all javascript library or plugin files.
    • SASS - contains all SCSS files.
    • index.html - contains all UI code.
  • Screenshot - Contains screenshot of template.
  • readme.txt - contains the information about Bootstrap notification design
